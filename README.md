# UpAI Skills

A collection of Claude Code skills by [UpAI](https://www.threads.com/@up.tech.ai).

## Git Worktrees

These skills provide worktree management as a temporary fix until Claude Code adds base branch settings for worktree creation natively.

- **worktree-new-temp** — Create a new git worktree and launch a Claude session inside it.

```
npx skills@latest add up-ai-code/skills/worktree-new-temp
```

- **worktree-remove-temp** — Remove an existing git worktree, with safeguards for uncommitted changes.

```
npx skills@latest add up-ai-code/skills/worktree-remove-temp
```
